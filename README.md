      # JRCS - Taller (scaffold)

      Este es un proyecto base generado para el taller **JRCS**. Contiene un scaffold mínimo con:
      - Lista de productos + carrito (componentes React).
      - Formulario de contacto con validación y plantilla para guardar en Firestore.
      - Configuración de Firebase (archivo de ejemplo).
      - Instrucciones rápidas.

      ## Cómo usar
      1. Copia este proyecto a tu máquina local.
      2. Instala dependencias:
         ```
         npm install
         ```
      3. Inicia en modo desarrollo:
         ```
         npm start
         ```
      4. Reemplaza `src/firebase/config.js` con tu configuración real de Firebase.

      ## Estructura

- src/
  - components/
  - firebase/
  - styles/
